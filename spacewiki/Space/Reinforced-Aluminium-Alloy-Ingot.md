# Obtaining

## Crafting

Reinforced Aluminium Alloy Ingots can be crafted using the following items:

* 1x [Palladium Ingot](Palladium-Ingot)
* 2x [Aluminium Alloy Ingot](Aluminium-Alloy-Ingot)
