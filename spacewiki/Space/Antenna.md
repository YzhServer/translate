An Antenna is a crafting component.

# Obtaining

## Crafting

An Antenna can be crafted using the following items:

* 1x [Plastic](Plastic)
* 1x [Surge Proof Wire](Surge-Proof-Wire)
* 1x [Capacitor](Capacitor)
* 2x [Reinforced Palladium Ingot](Reinforced-Palladium-Ingot)
